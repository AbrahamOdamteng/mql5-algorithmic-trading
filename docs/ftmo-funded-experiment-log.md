Use this log for funded-account survival, payout, drawdown, monthly return, scaling, and capital-preservation experiments. Funded-stage strategy may be different from challenge-stage strategy.

Do not use this file for challenge `+10%` or verification `+5%` pass-rate-first research. Use `ftmo-challenge-experiment-log.md` for challenge-stage work.

## Format

Each entry should include:

- Date
- Goal
- Change or experiment
- Test setup
- Outcome
